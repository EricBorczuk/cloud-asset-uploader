Cloud Asset Uploader API documentation

http://localhost:8080/

Request Signed Upload URL

/api/upload post

post

Request an upload via an S3 signed URL.

post /api/upload

Request an upload via an S3 signed URL.

  • Request
  • Response

Body

Media type: application/json

Type: object

Properties
  • object_key: required(string)

    The name of the file to be uploaded to S3

  • expires_in: (number)

    The amount of time (in seconds) that the signed URL will remain valid. Defaults to 60 seconds.

Example:

{
  "object_key": "my_cool_file.jpg",
  "expires_in": 600
}

HTTP status code 200

Body

Media type: application/json

Type: any

Example:

{
  "signed_info": {
    "url": "https://ericborczuk.s3.amazonaws.com/",
    "fields": {
      "key": "ABCDEF.txt",
      "AWSAccessKeyId": "XXXXXXXXXXXXXXXX",
      "policy": "SOME_LONG_POLICY",
      "signature": "SOME_SIGNATURE"
    }
  },
  "asset_id": 1
}

Update Asset Status

/api/status put

put

Updates the status of a provided asset.

put /api/status

Updates the status of a provided asset.

  • Request
  • Response

Body

Media type: application/json

Type: object

Properties
  • asset_id: required(number)

    The ID of the asset to update.

  • uploaded_status: required(string)

    The status to update. Valid values (pending|complete)

Example:

{
  "asset_id": 1,
  "uploaded_status": "complete"
}

HTTP status code 200

Body

Media type: application/json

Type: any

Example:

{
  "success": true,
  "uploaded_status": "complete"
}

Request Signed Download URL

/api/access get

get

Request a signed URL to access an asset. The asset must have an uploaded_status of 'complete'.

get /api/access

Request a signed URL to access an asset. The asset must have an uploaded_status of 'complete'.

  • Request
  • Response

Query Parameters

  • asset_id: required(number)

    The ID of the asset you want access to.

  • expires_in: (number)

    The amount of time (in seconds) that the signed URL will remain valid. Defaults to 60 seconds.

HTTP status code 200

Body

Media type: application/json

Type: any

Example:

{
  "url": "https://fancy.url.com",
  "asset_id": 1
}
